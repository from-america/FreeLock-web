2 Description of Service

FreeLock is a privacy-first app blocking application. It allows you to:

  • • Block selected apps using Apple's Screen Time framework
  • • Register NFC cards or tags as physical unlock keys
  • • Set emergency unlock methods (passcode + biometrics)
  • • Configure optional accountability contacts
  • • View local audit logs of lock/unlock events

All processing occurs on your device. No app blocking data is sent to any server.
