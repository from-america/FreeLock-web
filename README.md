# FreeLock Website

Privacy Policy, Terms of Service, and landing page for FreeLock iOS app.

## Contents
- `index.html` — Landing page
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Service
- `css/style.css` — Styling

## Deployment

The site is deployed to GitHub Pages. Updates to `main` branch automatically deploy via GitHub Actions workflow.

## About FreeLock

FreeLock is a privacy-first iOS app blocker that uses physical NFC cards as unlock keys. Learn more at [github.com/EricSpencer00/FreeLock](https://github.com/EricSpencer00/FreeLock).
